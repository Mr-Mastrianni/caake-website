Implementation Guide

Follow these steps to implement the Zed-Style 3D animations and UI components on your website.

  1. Include Required Libraries

    Add Three.js and Vanta.js to your project:

    <script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/vanta@latest/dist/vanta.globe.min.js"></script>
  2. Include the Zed-Style CSS 
    <link rel="stylesheet" href="assets/css/zed-style.css">
  3. Import the ZedStyleAnimation Class 
    import ZedStyleAnimation from './assets/js/three-animations/zed-style-animation.js';
  4. Create a Container for the Animation 
    <div id="zed-animation-container" class="animation-container"></div>
  5. Initialize the Animation 
    // Initialize when the DOM is loaded
document.addEventListener('DOMContentLoaded', () => {
    const zedAnimation = new ZedStyleAnimation('zed-animation-container', {
        // Customize options as needed
        backgroundColor: 0x0a0a12,
        wireframeColor: 0x5773ff,
        glowColor: 0x00f0ff,
        accentColor: 0xff007a,
        bloomStrength: 1.5,
        rotationSpeed: 0.005,
        vantaEffect: 'globe'
    });
});
  6. Add UI Components

    Use the provided CSS classes to create futuristic UI elements:

    <div class="holographic-card">
    <h3 class="neon-text">Card Title</h3>
    <p>Card content goes here.</p>
    <a href="#" class="zed-button">Action</a>
</div>
