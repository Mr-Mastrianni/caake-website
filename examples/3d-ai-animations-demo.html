How to Use These Animations

To add these animations to your website, follow these steps:

  1. Add the data-animation attribute to any container element with one of these values:
    • ai-brain - AI Brain Visualization
    • machine-learning-cube - Machine Learning Cube
    • ai-voice-waveform - AI Voice Waveform
    • neural-network - Neural Network
    • workflow-automation - Workflow Automation
    • digital-twin - Digital Twin
    • data-visualization - Data Visualization
  2. Make sure the container has a position of relative or absolute and a defined height.
  3. The animation will automatically be initialized when the page loads.
<!-- Example HTML -->
<div class="hero-section" style="position: relative; height: 500px;">
  <div data-animation="ai-brain"></div>
  <div class="hero-content">
    <h1>AI-Powered Solutions</h1>
    <p>Transform your business with our cutting-edge AI technology</p>
  </div>
</div>
