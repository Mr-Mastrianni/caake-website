Background Logo

The logo can also be used as a background element with reduced opacity.

<div id="background-logo" class="logo-3d-container logo-fullscreen"></div>

// CSS
.logo-fullscreen {
    position: fixed;
    top: 0;
    left: 0;
    width: 100vw;
    height: 100vh;
    z-index: -1;
    opacity: 0.3;
}
